Pagaya - Web Scraping Home Exam

This is a readme file for running the home_exam files.

The folder contains:

1. pagaya_home_test_data_collection_part1 - for transforming the graph of demographics
and capture it in a table format.

2. pagaya_home_test_data_collection_part2 - for extracting the population value and
percentage per gender and capturing it in a table format.

3. pagaya_home_test_api - for the second part in the exam using CDX API

4. zc_list - for getting all the US Zipcodes in the given website.

requirements.txt included for running all the scripts.
